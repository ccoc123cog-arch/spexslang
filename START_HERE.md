### 2. Попробуйте этот код:
```spexs
# Ваша первая программа на Spexs!
var name = "Программист"
print("Привет, " + name + "!")

func factorial(n) {
    if (n <= 1) {
        return 1
    }
    return n * factorial(n - 1)
}

print("Факториал 5 = " + factorial(5))

var numbers = [1, 2, 3, 4, 5]
var sum = 0
for num in numbers {
    sum = sum + num
}
print("Сумма чисел: " + sum)
```

## 🎯 Возможности языка Spexs:

### ✅ Работает прямо сейчас:
- Переменные: `var`, `const`
- Функции: `func`, `return`
- Условия: `if`, `else`, `elif`
- Циклы: `while`, `for...in`
- Операторы: `+`, `-`, `*`, `/`, `%`, `==`, `!=`, `<`, `>`, `and`, `or`, `not`
- Типы: числа, строки, булевы, списки, null
- Рекурсия
- Комментарии: `#`

### 🔜 Планируется:
- Классы и объекты
- Импорт модулей
- Горутины (как в Go)
- Каналы
- И многое другое!

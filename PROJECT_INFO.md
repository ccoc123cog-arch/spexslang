# 🚀 Spexs Programming Language - Информация о проекте

## 📋 Описание

**Spexs** - это современный язык программирования, созданный с целью объединить лучшие черты трех популярных языков:

- 🐍 **Python** - простота и читаемость синтаксиса
- ☕ **Java** - надежность и структурированность
- ⚡ **Go** - производительность и параллелизм

## 🎯 Цели проекта

1. Создать простой в изучении язык программирования
2. Обеспечить высокую производительность
3. Поддержать современные парадигмы программирования
4. Предоставить удобные инструменты для разработчиков

## 📦 Что включено

### 1. Веб-сайт (ASP.NET Core)
- **Главная страница** - презентация языка
- **Playground** - онлайн-интерпретатор для тестирования кода
- **Страница загрузки** - информация о скачивании и установке
- **Адаптивный дизайн** - работает на всех устройствах

### 2. Интерпретатор Spexs
Полностью функциональный интерпретатор, написанный на C#:

#### Lexer (Лексический анализатор)
- Токенизация исходного кода
- Распознавание ключевых слов
- Обработка операторов и литералов
- Поддержка комментариев

#### Parser (Синтаксический анализатор)
- Построение AST (Abstract Syntax Tree)
- Проверка синтаксиса
- Обработка приоритета операторов
- Поддержка вложенных структур

#### Interpreter (Интерпретатор)
- Выполнение кода
- Управление переменными и функциями
- Обработка рекурсии
- Вычисление выражений

### 3. Примеры кода (.sxp)
- `hello.sxp` - Hello World
- `functions.sxp` - Работа с функциями
- `loops.sxp` - Циклы и итерации
- `calculator.sxp` - Простой калькулятор
- `fibonacci.sxp` - Числа Фибоначчи
- `guess-number.sxp` - Игра "Угадай число"
- `prime-numbers.sxp` - Поиск простых чисел
- `sorting.sxp` - Сортировка (в разработке)

✅ КОМАНДА УСТАНОВЛЕНА!

Теперь можно запускать файлы Spexs из любой папки:

┌──────────────────────────────────────────────────────────────┐
│ ЗАПУСК ФАЙЛА:                                                │
│                                                              │
│   spexs hello.sxp                                            │
│   spexs examples/factorial.sxp                               │
│   spexs path/to/program.sxp                                  │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ ИНТЕРАКТИВНЫЙ РЕЖИМ (REPL):                                  │
│                                                              │
│   spexs                                                      │
│                                                              │
│   >>> print("Hello!")                                        │
│   Hello!                                                     │
│   >>> var x = 5 + 3                                          │
│   >>> print(x)                                               │
│   8                                                          │
│   >>> exit                                                   │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ ПРИМЕРЫ:                                                     │
│                                                              │

└──────────────────────────────────────────────────────────────┘

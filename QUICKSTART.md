### Пример кода для тестирования:

```spexs
# Простая программа
var name = "Мир"
print("Привет, " + name + "!")

func factorial(n) {
    if (n <= 1) {
        return 1
    }
    return n * factorial(n - 1)
}

print("Факториал 5: " + factorial(5))

var numbers = [1, 2, 3, 4, 5]
var sum = 0
for num in numbers {
    sum = sum + num
}
print("Сумма: " + sum)
```

## 📝 Возможности языка Spexs:

### ✅ Реализовано:
- Переменные (`var`, `const`)
- Функции (`func`)
- Условия (`if`, `else`, `elif`)
- Циклы (`for`, `while`)
- Операторы (+, -, *, /, %, ==, !=, <, >, <=, >=, and, or, not)
- Типы данных (числа, строки, булевы, списки, null)
- Комментарии (`#`)
- Рекурсия

### 🔜 Планируется:
- Классы и объекты
- Импорт модулей
- Горутины (как в Go)
- Каналы (chan)
- Defer
- Обработка ошибок
- Стандартная библиотека
